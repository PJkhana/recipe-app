This is a full-stack recipe management application built with:

- **Frontend**: Angular (standalone components, Angular Router)
- **Backend**: ASP.NET Core Web API
- **Database**: MySQL using Entity Framework Core

## Features

- Add, Edit, Delete recipes
- View recipe details
- Form validation
- Responsive UI

### Frontend

npm install
ng serve
